# homeleyu-journal

## 项目简介

`homeleyu-journal` 是一个用于归档和发布独立 HTML 页面的仓库。这里不针对任何特定的域名或网站，仅用于存放各类静态 HTML 文件，方便长期保存与分享。

## 目录说明

```
homeleyu-journal/
├── index.html          # 仓库首页/导航页
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下按时间或主题存放独立的 HTML 文件。
- `assets/` 目录存放页面共用的样式表、脚本及图片资源。

## 页面归档说明

本仓库中的页面均为独立的 HTML 文件，无需后端支持，可直接通过浏览器打开。每个页面代表一次独立的记录或展示，内容涵盖笔记、日记、项目展示等。

- 所有页面均为纯前端实现。
- 不依赖外部 API 或动态数据源。
- 页面之间相互独立，无强制关联。

## 维护说明

- 新增页面请放入 `pages/` 目录，并更新 `index.html` 中的导航链接。
- 请保持文件命名简洁，建议使用英文小写与短横线分隔。
- 如使用公共资源，请放置于 `assets/` 对应子目录，避免重复引入。
- 提交时请附带简要说明，描述新增或修改的内容。
